Welcome to MyFlix API Documentation

This API provides access to information about movies, directors, and genres. Users can interact with the API to retrieve data, update their profiles, and manage their list of favorite movies.

MyFlix API Endpoints

Models Method Type Endpoint URL Description Request Format Response Format
Movies GET /movies Returns a list of all movies. N/A JSON array of movie objects
Movies GET /movies/:title Returns data about a single movie based on the title. N/A JSON object containing movie details
Movies GET /movies/genre/:genreName Returns data about a genre based on the name. N/A JSON array of movie objects
Movies GET /movies/directors/:directorName Returns data about a director based on the name. N/A JSON array of movie objects
Users POST /users Allows a new user to register. JSON object with user details
(Example: { "Username": "newUser", "Password": "password123", "Email": "newUser@example.com", "Birthday": "2000-01-01" })		 JSON object containing user details
Users PUT /users/:Username Allows users to update their user information. JSON object with updated user details
(Example: { "Username": "updatedUser", "Password": "updatedPassword", "Email": "updatedEmail@example.com", "Birthday": "1990-01-01" })		 JSON object containing updated user details
Users PATCH /users/:Username/movies/:MovieID Allows users to add a movie to their favorites list. N/A JSON object containing user details
Users DELETE /users/:Username/movies/:MovieID Allows users to remove a movie from their favorites list. N/A JSON object containing user details
Users DELETE /users/:Username Allows existing users to delete their accounts. N/A Text message (success or error message)
Users GET /users Returns a list of all users. N/A JSON array of user objects
Users GET /users/:Username Returns data about a single user based on the username. N/A JSON object containing user details
